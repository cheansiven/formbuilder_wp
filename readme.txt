== Description ==
Create a form without knowing any HTML or PHP!

Drag and drop form elements.

Manage multiple forms in a few clicks away, and way more flexible and User Friendly than other form plugins.

Easy to use user interface.

Send submitted form data via email.

Dynamically set form values from the URL.

Preview while building.

14 included element types.

100% spam protected.

Ajax Powered.

WYSIWIG Mailing system.

Shortcode Supported.

Forms can inherit themes, style without any issue.

Redirect URL.

Custom autoreply.


== Installation ==

1. Upload the entire first-contact-form folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'First Contact' menu in your WordPress admin panel.
